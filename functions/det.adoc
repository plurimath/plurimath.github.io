== Det

=== AsciiMath

[source,asciimath]
----
det(x)
----


=== LaTeX

[source,latex]
----
\det{( x )}
----


=== MathML

[source,xml]
----
<math xmlns="http://www.w3.org/1998/Math/MathML" display="block">
  <mstyle displaystyle="true">
    <mrow>
      <mi>det</mi>
      <mrow>
