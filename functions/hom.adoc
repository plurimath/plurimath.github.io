[[hom]]
== Hom

=== LaTeX

[source,latex]
----
\hom{( x )}
----


=== UnicodeMath

[source,unicodemath]
----
hom(x)
----


NOTE: In other conversions, the value of the class will be processed as a simple text. For example, the output of this example in *AsciiMath* will be represented as `hom (x)`.
