== Color


=== AsciiMath

[source,asciimath]
----
color(red)(y)
----



=== LaTeX

[source,latex]
----
{\color{red} y}
----



=== MathML

[source,xml]
----
<math xmlns="http://www.w3.org/1998/Math/MathML" display="block">
  <mstyle displaystyle="true">
    <mstyle mathcolor="red">
      <mi>x</mi>
    </mstyle>
  </mstyle>
</math>
----
